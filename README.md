# Sales Performance Analysis Using SQL

## Project Overview

This project analyzes sales data using SQL to identify business insights related to sales performance, profitability, 
customer behavior, product performance, and regional trends.
The goal of this project is to practice SQL skills and answer real-world business questions using data analysis techniques.

## Dataset

Dataset: Sample Superstore Dataset

The dataset contains information about:

* Orders
* Customers
* Products
* Categories
* Regions
* Sales
* Profit
* Quantity
* Discount

## Tools Used

* SQL
* MySQL (or your SQL database)
* GitHub

## SQL Concepts Used

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* Aggregate Functions
* DISTINCT
* LIMIT
* Date Functions

## Business Questions Answered

1. What is the total sales revenue?
2. What is the total profit?
3. Which products generate the highest sales?
4. Which products generate the highest profit?
5. Which customers contribute the most revenue?
6. Which region performs best?
7. Which category generates the highest sales?
8. What are the monthly sales trends?
9. What are the yearly sales trends?
10. Which cities generate the most revenue?

## Key Insights

* Technology category generated the highest sales.
* The West region contributed the highest revenue.
* A small number of customers generated a significant portion of sales.
* Monthly sales trends revealed seasonal patterns.
* Some products generated high sales but low profit.
